# SolCam

SolCam is a professional multi-camera security dashboard that leverages the power of AI detection to provide enhanced surveillance capabilities. Built using **PyQt6**, **OpenCV**, and **YOLOv8**, SolCam offers a user-friendly interface and real-time monitoring features.

## Features
- Multi-camera support
- AI-powered motion detection
- Real-time video processing
- User-friendly interface

## Technologies Used
- **PyQt6** for the graphical user interface
- **OpenCV** for computer vision tasks
- **YOLOv8** for object detection

## Installation
To set up SolCam, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Max6079/SolCam.git
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python main.py
   ```
